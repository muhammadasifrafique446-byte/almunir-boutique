## 🆘 Need Help?

**Images not showing?**
- Check images are in `images/` folder
- Check file names are exactly: `design1.jpg`, `design2.jpg`, etc.
- Check file format is JPG or PNG

**WhatsApp button not working?**
- Make sure you updated the phone number
- Start with: `https://wa.me/+92`
- Your number should be 12 digits total (92 + 10 digits)

**Website not live on GitHub?**
- Check repo name is: `boutique`
- Check Settings → Pages → Source is `main` branch
- Wait 2-3 minutes for deployment
